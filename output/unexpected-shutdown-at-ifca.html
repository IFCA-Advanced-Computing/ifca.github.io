Unexpected shutdown at IFCA

The IFCA datacenter suffered an unexpected shutdown, caused by a power cut due to a malfunctioning electrical installation. We are working hard on restoring the situation.

Please visit this page for more updates about the recovery process.

  • 2014-05-15 14:32 CEST Grid Cluster is fully available.

  • 2014-05-15 13:17 CEST GPFS storage is fully available.

    All filesystems are now available.

  • 2014-05-15 12:44 CEST GPFS storage partially available.

    /gpfs/gaes filesystem is not available.

  • 2014-05-15 12:44 CEST GRIDUI access is available.

    GRIDUI cluster has been partially restored. There's no access to the batch system though.

  • 2014-05-15 12:00 CEST Cloud infrastructure is recovered.

    portal.cloud.ifca.es is now fully functional.

  • 2014-04-14 18:00 CEST Power cut.

    Due to a malfunctioning electrical installation the IFCA datacenter suffered an unexpected power cut. Only critical systems were restored.
